
ModCommon
^^^^^^^^^

ModCommon is a toolset designed to make creating Hollow Knight mods easier. It includes prebuilt components which you can attach to gameobjects to modify them, Extenstions to common Unity and Hollow Knight classes to improve their functionality, and custom Hooks which can help you run your code at arbitrary times.
